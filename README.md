## Compilation Instructions

this project uses **maven** and **JavaFX**

### requirements

Java 21 <br>
Maven wrapper (included in the repo)<br>
JavaFx (done by maven)

## how to build

run this in the terminal inside the project directory:<br>
`./mvnw.cmd clean compile`

## how to run

`./mvnw.cmd javafx:run`
